# Login

**URL** : `api/auth/login`

**Method** : `POST`

**Data example**
```
{
"username": "iloveauth@example.com",
"password": "abcd1234"
}
```
# Register

**URL** : `api/auth/register`

**Method** : `POST`

**Data example**
```
{
"username": "enes",
"email": "enes@enes.com",
"password": "abcd1234"
}
```
